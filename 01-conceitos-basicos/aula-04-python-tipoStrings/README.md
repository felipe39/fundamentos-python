# 🐍 Aula 04 - Tipos de Dados: Strings

## 🎯 Objetivo

Aprender a manipular textos utilizando o tipo `str` em Python.

## 📚 Conteúdos estudados

- O que é uma String (`str`)
- Aspas simples e duplas
- Concatenação (`+`)
- Repetição (`*`)
- Comprimento com `len()`
- Índices
- Índices negativos
- Fatiamento (Slice)

## 📂 Arquivos

- exemplo_strings.py
- exercicio01.py
- exercicio02.py

## ✅ Aprendizados

Ao concluir esta aula aprendi a:

- Criar variáveis do tipo String.
- Juntar textos utilizando concatenação.
- Repetir Strings.
- Descobrir o tamanho de um texto.
- Acessar caracteres utilizando índices.
- Utilizar índices negativos.
- Realizar fatiamento de Strings.
