🧮 Aula 05 - Média Ponderada

🎯 Objetivo

Aprender a receber dados digitados pelo usuário utilizando "input()", converter valores utilizando "float()" e realizar cálculos matemáticos em Python.

📚 Conteúdos estudados

- Utilização da função "input()".
- Conversão de dados utilizando "float()".
- Operações matemáticas.
- Média ponderada.
- Utilização do "print()".
- Arredondamento de valores com "round()".
- Organização da saída do programa utilizando "\n".

📂 Arquivos

- "exemplo_media.py"
- "exercicio01.py"

✅ Aprendizados

Ao concluir esta aula fui capaz de:

- Solicitar informações ao usuário.
- Trabalhar com números decimais.
- Realizar cálculos utilizando variáveis.
- Calcular médias ponderadas.
- Exibir resultados formatados na tela.
- Melhorar a apresentação das mensagens do programa.

🚀 Resumo

Esta aula foi importante para compreender como criar programas interativos. Pela primeira vez, o usuário fornece dados ao programa, que realiza cálculos e apresenta um resultado final de maneira organizada.
