📚 Aula 07 – Listas e Tuplas em Python

🎯 Objetivos da Aula

Nesta aula aprendi a:

- Criar Listas e Tuplas.
- Entender a diferença entre estruturas mutáveis e imutáveis.
- Acessar elementos utilizando índices.
- Adicionar, remover e ordenar elementos em listas.

---

🧠 O que são Tuplas?

As Tuplas são utilizadas quando as informações não precisam ser alteradas.

Exemplo:

meses = (
    "Janeiro",
    "Fevereiro",
    "Março"
)

Características

- Utilizam "()"
- São imutáveis.
- Permitem acessar informações através de índices.
- São úteis quando os dados não devem ser modificados.

«💡 Pense nas Tuplas como um cofre: você pode visualizar o que está dentro, mas não pode alterar o seu conteúdo.»

---

🧠 O que são Listas?

As Listas permitem adicionar, remover, modificar e organizar informações.

Exemplo:

alunos = [
    "João",
    "Maria",
    "Pedro"
]

Características

- Utilizam "[]"
- São mutáveis.
- Permitem adicionar, remover e alterar elementos.
- São uma das estruturas mais utilizadas em Python.

«💡 Pense nas Listas como um caderno: você pode escrever, apagar, modificar e reorganizar tudo o que estiver nele.»

---

📌 Principais conceitos aprendidos

Contando elementos

len(lista)

Adicionando elementos

lista.append("Felipe")

Inserindo em posições específicas

lista.insert(1, "Felipe")

Removendo elementos

lista.pop(1)

ou

lista.remove("Felipe")

Ordenando listas

lista.sort()

Concatenando listas

lista1 + lista2

Utilizando índices

lista[0]

Fatiamento (Slicing)

texto[3:5]
